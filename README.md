# FM Remix Forge Studio V85 REAL APP BASE

Cette version n'est pas une coquille vide. Elle contient une application React/Vite utilisable dans le navigateur et packagée Electron.

## Fonctions réelles

- Import audio local
- Décodage Web Audio API
- Waveform canvas
- Analyse BPM approximative
- Estimation simple tonalité
- Choix style : Kompa, Gouyad, Afro, Amapiano, Dancehall, House
- Génération remix simple : drums + basse + accords + hook
- Lecture remix
- Mix optionnel avec la source originale
- Export WAV
- Workflow GitHub Windows

## Commandes

```bash
npm install
npm run check
npm run build
npm run dist:win
```
